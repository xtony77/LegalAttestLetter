# 台灣存證信函產生器

免費線上存證信函產生器，自動排版產生符合郵局格式的 PDF。

所有資料完全在瀏覽器處理，不會上傳至任何伺服器。

## 展示

[展示網頁](https://legal-attest-letter.xtony77.workers.dev/)

## 功能

- 填寫寄件人、收件人、副本收件人與信函內文，一鍵產生 PDF
- 支援多位寄件人／收件人／副本收件人
- 自動套用郵局存證信函格式排版
- 表單驗證與錯誤提示
- 響應式設計，手機與桌面皆可使用

## 技術

- React + TypeScript
- Vite
- Tailwind CSS
- pdf-lib（PDF 產生）

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```
